# Calendario de Exámenes

App local para planificar meses concretos y registrar exámenes con dificultad baja, media o alta.

## Usar

Abre `index.html` en el navegador.

Los datos se guardan en `localStorage`, así que se mantienen en el mismo navegador aunque cierres la página.

## Funciones

- Añadir los meses que quieras ver, por ejemplo marzo y abril.
- Ver cada mes completo en formato calendario.
- Añadir exámenes con fecha y hora desde cualquier día o desde el botón superior.
- Indicar créditos de la asignatura: 3 o menos verde, de 4 a 7 amarillo, 8 o más rojo.
- Ver cuántos días quedan para cada examen.
- Editar o eliminar exámenes pulsando sobre ellos.
